# MyFolio
from [PSD](https://dribbble.com/shots/1614563-Free-PSD-for-Single-Page-Portfolio) to [HTML](https://manny-folio.herokuapp.com/) 

## Features
* Responsive design
* FlexSlider
* WOWJS
* ScrollSpy's Bootstrap
* SweetAlerts

## Tested and it works in:
* Chrome
* Opera Next
* FireFox
* Safari (ipad y desktop)
* IE10
